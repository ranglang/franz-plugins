使用方法

1. 下载

   ​	

[下载文件架](https://codeload.github.com/ranglang/franz-plugins/zip/master)

```
git clone git@github.com:ranglang/franz-plugins.git
```

2. 一键安装

```
sh ./install.sh

```

3. 重启Franz

注意Franz默认点击关闭后在后台运行，你可能需要手动结束进程
